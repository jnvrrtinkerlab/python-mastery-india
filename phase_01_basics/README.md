## Key Concepts

### print() - Display Output
```python
print("Hello, World!")
print("Learning Python")
```

### Variables - Store Values
```python
name = "Karthik"
age = 15
marks = 95.5
```

### Data Types
- `int` - Integers (whole numbers): 5, 10, -3
- `float` - Decimals: 3.14, 9.8  
- `str` - Text (strings): "Hello", 'Python'
- `bool` - True or False

### Operators
- Arithmetic: +, -, *, /, %, **
- Comparison: ==, !=, >, <, >=, <=
- Logical: and, or, not

### input() - Get User Data
```python
name = input("Enter your name: ")
age = int(input("Enter your age: "))
```
