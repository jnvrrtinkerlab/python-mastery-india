# Contributing to Python Mastery India

## We Love Your Input!

We're excited that you're interested in contributing to Python Mastery India. Whether it's:
- Reporting bugs
- Suggesting improvements
- Adding new lessons
- Fixing typos
- Improving documentation

All contributions are welcome!

## How to Contribute

### 1. Fork the Repository
```bash
git clone https://github.com/YOUR_USERNAME/python-mastery-india.git
cd python-mastery-india
```

### 2. Create a Branch
```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes
- Keep commits clear and descriptive
- Follow Python PEP 8 style guide
- Add comments to explain complex code
- Include Indian examples where possible

### 4. Test Your Changes
```bash
python -m pytest  # If applicable
```

### 5. Push to Your Fork
```bash
git push origin feature/your-feature-name
```

### 6. Submit a Pull Request
Our team will review and merge your contribution!

## Content Guidelines

### For Lesson Contributions:
- Use Indian names and contexts (Lahari, Karthik, Bhavesh, etc.)
- Include 3-5 practice problems with solutions
- Add a mini-project at the end
- Use clear, beginner-friendly language
- Include code examples and output

### Code Style:
- Follow PEP 8
- Use meaningful variable names
- Add comments for complex logic
- Test your code before submitting

## Reporting Issues

Found a bug? Please open an issue with:
1. Description of the problem
2. Steps to reproduce
3. Expected vs actual behavior
4. Python version and OS

## Questions?

Feel free to open an issue or contact us at yeshwanth@example.com
