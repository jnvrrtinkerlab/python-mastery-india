## Phase 1: Python Basics (20-30 hours)

### Topics:
1. **Variables & Print Statements**
   - Creating variables
   - Naming conventions
   - Print function basics
   - Example: `name = "Raj"; print(f"Hello, {name}!")`

2. **Data Types**
   - Integers (int)
   - Floats (float) 
   - Strings (str)
   - Booleans (bool)
   - Type checking with type()
   - Indian context: Storing student marks, roll numbers, fees in rupees

3. **Operators**
   - Arithmetic: +, -, *, /, //, %, **
   - Comparison: ==, !=, <, >, <=, >=
   - Logical: and, or, not
   - Example: `10 % 3` (remainder in Indian currency calculations)

4. **Taking User Input**
   - input() function
   - Converting input data types
   - Example: `age = int(input("Enter your age: "))`

## Running the Lessons

### Basic Python Files:
```bash
python phase_01_basics/lesson_01_hello_world.py
```

### With Dependencies:
```bash
pip install -r requirements.txt
python phase_08_game_development/snake_game.py
```

## Troubleshooting

- **Import errors**: Install requirements.txt
- **Syntax errors**: Check Python version (3.8+)
- **Permission errors**: Run with appropriate privileges
- **Module not found**: Install missing package via pip
